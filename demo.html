Wikipedia-tooltip for jQuery

jQuery plugin: Wikipedia-tooltip

Current version: 0.1
License: MIT/GPL

Dependencies:
jQuery (tested with version 1.7.1)

Demo:
mySQL (hover over me)
Albert Einstein (hover over me)

Code

<head>
<script type='text/javascript'
src='http://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js'></script>
<script type='text/javascript' src='js/wikipedia-tooltip.0.1.js'></script>
<link rel="stylesheet" type="text/css" media="all" href="js/wikipedia-tooltip.0.1.css" />

<script type="text/javascript">
  $(document).ready(function(){
  $('.wikihover').wiki_tooltip({wiki_lang : 'en'});
  });
  </script>
</head>

<body>
<abbr class="wikihover" rel="mySQL">mySQL (hover over me)</abbr><br><br>
<abbr class="wikihover" rel="Albert Einstein">Albert Einstein (hover over me)</abbr>
</body>
